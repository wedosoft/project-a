Explore agent 결과를 검증합니다.

## 검증 절차

1. $ARGUMENTS에 지정된 파일(또는 관련 파일)을 Read 도구로 직접 열기
2. agent가 주장한 내용이 실제 코드에 있는지 라인별로 확인
3. 불일치 발견 시 재탐색 후 정확한 결과 보고
4. 일치 시 "파일경로:라인번호 - 확인됨"으로 보고

## 규칙
- "있을 것 같다"가 아닌 "있다 / 없다"로만 결론
- 라인 번호 없이 결론 내리지 말 것
- 모든 핵심 주장은 직접 읽은 코드로 뒷받침할 것
