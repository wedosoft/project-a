# 미드저니 카드뉴스 배경 생성기

600x600 카드뉴스용 미드저니 프롬프트를 자동으로 생성하는 Claude 스킬입니다.

## 사용 방법

주제만 말씀해주시면 최적화된 미드저니 프롬프트를 생성해드립니다.

### 예시

**사용자**: "재테크 팁 카드뉴스 배경 만들어줘"

**Claude**: 3가지 스타일 옵션을 제공합니다:
1. 전문적이고 신뢰감 있는 금융 스타일
2. 친근하고 접근하기 쉬운 스타일  
3. 프리미엄하고 고급스러운 스타일

각 스타일마다 미드저니에서 바로 사용할 수 있는 영문 프롬프트가 제공됩니다.

## 지원하는 주제

- **비즈니스/테크**: 스타트업, 마케팅, 기업
- **건강/웰니스**: 운동, 명상, 영양, 의료
- **금융**: 투자, 저축, 암호화폐, 부동산
- **교육**: 온라인 학습, 언어, 어린이 교육
- **라이프스타일**: 여행, 음식, 패션, 연애
- **엔터테인먼트**: 음악, 영화, 게임
- **환경**: 지속가능성, 날씨
- **계절별**: 봄, 여름, 가을, 겨울
- **문화**: 한국 전통, 미니멀
- **트렌드**: Y2K, 코티지코어, 다크 아카데미아

## 특징

✅ 텍스트 오버레이에 최적화  
✅ 600x600 (1:1) 비율 지원  
✅ 주제별 맞춤 색상 팔레트  
✅ 3-5가지 스타일 옵션 제공  
✅ 한국어 입력, 영문 프롬프트 출력  
✅ Midjourney v6 최적화

## 프롬프트 구조

생성되는 모든 프롬프트는 다음 요소를 포함합니다:

1. **주제 설명**: 명확하고 구체적인 비주얼 요소
2. **스타일 키워드**: 미니멀, 모던, 그라데이션 등
3. **색상 팔레트**: 구체적인 색상명 (예: soft blue, coral pink)
4. **분위기/질감**: 부드러운, 역동적인, 깔끔한 등
5. **기술 설정**: `--ar 1:1 --v 6` (필수)

## 사용 팁

**좋은 요청 예시**:
- "건강한 식습관 관련 카드뉴스 배경"
- "스타트업 채용 공고용 배경"
- "봄 시즌 여행 추천 카드뉴스"
- "암호화폐 투자 팁 배경 이미지"

**피해야 할 것**:
- 특정 브랜드 로고 요청
- 너무 복잡한 장면 묘사
- 텍스트 영역을 방해하는 바쁜 패턴

## 텍스트 오버레이 최적화

모든 배경은 텍스트 추가를 고려하여 생성됩니다:

- **중앙 60%**: 상대적으로 균일한 영역
- **가장자리**: 시각적 흥미를 위한 복잡한 요소 배치
- **그라데이션**: 텍스트 가독성을 위한 부드러운 전환
- **명암 대비**: 텍스트가 돋보이는 적절한 대비

## 자주 묻는 질문

**Q: 한국어로 요청해도 되나요?**  
A: 네! 한국어로 주제를 말씀하시면 영문 프롬프트를 생성해드립니다.

**Q: 프롬프트를 수정할 수 있나요?**  
A: 네, 원하는 색상이나 스타일을 구체적으로 말씀해주시면 맞춤 조정해드립니다.

**Q: 여러 옵션을 받을 수 있나요?**  
A: 기본적으로 3가지 스타일 옵션을 제공하며, 더 많은 옵션이 필요하면 요청하세요.

**Q: Midjourney v7도 지원하나요?**  
A: 현재는 v6를 기준으로 하며, 새 버전이 나오면 업데이트됩니다.
