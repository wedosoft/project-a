Utils

🛠️ 유틸리티 함수 모음 - 범용 헬퍼 함수 라이브러리

This module provides general-purpose utility functions used throughout the app. Designed with reusability and stability in mind, all functions guarantee error safety.

이 모듈은 앱 전반에서 사용되는 범용적인 유틸리티 함수들을 제공합니다. 재사용성과 안정성을 고려하여 설계되었으며, 모든 함수는 에러 안전성을 보장합니다.
